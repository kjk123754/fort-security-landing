# ФОРТ — лендинг систем відеонагляду та охорони

Односторінковий адаптивний сайт українською для компанії «ФОРТ», що встановлює системи відеонагляду та охорони. Сучасна синьо-чорно-біла палітра, геро-блок зі слайдером фону, дві колонки об'єктів, перелік послуг, анімовані переваги та завершальний CTA з формою.

## Структура
- `index.html` — розмітка сторінки та якірна навігація.
- `style.css` — стилі, сітки та адаптивні брейкпоінти.
- `script.js` — слайдер фонового зображення та поява карток переваг при скролі.
- `assets/logo-fort.svg` — оновлений логотип у синьо-біло-чорній палітрі.

## Перегляд локально
1. Встановіть будь-який простий HTTP-сервер або використайте вбудований Python:
   ```bash
   python3 -m http.server 8000
   ```
2. Відкрийте у браузері [http://localhost:8000](http://localhost:8000) і перейдіть до `index.html`.

Альтернативно, можна просто відкрити `index.html` у браузері як файл, але запуск через локальний сервер коректніше обробляє зовнішні ресурси та шрифти.

## Оновлення з GitHub
Репозиторій не має доданого віддаленого джерела. Щоб підтягнути зміни з GitHub або надіслати свої:

1. Додайте remote із вашою адресою сховища (HTTPS або SSH):
   ```bash
   git remote add origin https://github.com/<користувач>/<репозиторій>.git
   # або
   git remote add origin git@github.com:<користувач>/<репозиторій>.git
   ```
2. Перевірте, що remote зʼявився:
   ```bash
   git remote -v
   ```
3. Отримайте останні зміни та злийте їх у поточну гілку:
   ```bash
   git fetch origin
   git merge origin/main   # або використайте git pull origin main
   ```
4. Після внесення власних правок завантажте їх на GitHub:
   ```bash
   git push origin main    # замініть main на потрібну гілку, якщо інша
   ```
