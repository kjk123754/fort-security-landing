# ФОРТ — лендинг систем безпеки

Односторінковий адаптивний сайт українською мовою для компанії ФОРТ. Тема — синій/білий/чорний з акцентом на безпеку та конверсію.

## Структура сторінки
- Hero з формою заявки та слайдером фонів.
- "Наші об’єкти" у двох колонках (відеонагляд та охоронні системи).
- "Наші послуги" з 4 картками.
- "Чому нас обирають" з анімованими перевагами.
- Контактний блок із формою, телефоном та іконками месенджерів.

## Запуск локально
```bash
python3 -m http.server 8000
# відкрийте http://localhost:8000 у браузері
```

## Файли
- `index.html` — розмітка секцій та форм.
- `style.css` — стилі, сітки, анімації та адаптивність.
- `script.js` — слайдер hero та анімація появи переваг.

> Примітка: фон HERO та картки об’єктів використовують відкриті зображення з Unsplash.
